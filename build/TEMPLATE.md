# react-gh2project
> React github pages to project.

## installation
```shell
npm install -S @feizheng/react-gh2project
```

## update
```shell
npm update @feizheng/react-gh2project
```

## properties
__GENERATE_DOCS__

## usage
1. import css
  ```scss
  @import "~@feizheng/react-gh2project/dist/style.scss";

  // customize your styles:
  $react-gh2project-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
